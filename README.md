# 🏥 Hospital Management System

## 📌 Project Details

This is a simple web-based Hospital Management System developed as a college project.
It is designed to manage basic hospital activities in an organized way using a database.

The system allows users to store, update, and manage hospital data such as patients, doctors, and appointments.

The project is built using:

* HTML, CSS, JavaScript (Frontend)
* Python Flask (Backend)
* MySQL with XAMPP (Database)

---

## 🚀 Features & Operations

### 👨‍⚕️ Patient Management

* Add new patient details
* View patient records
* Update patient information
* Delete patient data

---

### 🩺 Doctor Management

* Add doctor details
* View list of doctors
* Update doctor information
* Delete doctor records

---

### 📅 Appointment Management

* Book appointments between patient and doctor
* View all appointments
* Update appointment details
* Cancel appointments

---

### 🏥 Database Operations

* Store all data in MySQL database
* Perform CRUD operations:

  * Create (Insert data)
  * Read (View data)
  * Update (Edit data)
  * Delete (Remove data)

---

### 💻 System Operations

* Connect frontend with backend using Flask
* Handle user requests and responses
* Display data dynamically on web pages

### ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/AryanFultariya271107/Hospital-management-system.git
cd hospital-management-system
```

---

### 2. Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

### 3. Start XAMPP

* Open XAMPP Control Panel
* Start **Apache** and **MySQL**

---

### 4. Setup Database

1. Open browser:

```
http://localhost/phpmyadmin
```

2. Create a new database:

```
hospital_mgmt
```

3. Import SQL file:

* Click **Import**
* Select file: `hospital.sql`
* Click **Go**

---

### 5. Run the Application

```bash
python app.py
```

---

### 6. Open in Browser

```
http://127.0.0.1:5000
```
